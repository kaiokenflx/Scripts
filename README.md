# Collection of useful helper scripts

### Proxmox
- Migration ESXi to Proxmox - `vmdk2qcow2.sh`: Convert VMDK image files to QCOW2 virtual disk image format
- `ve_import_qcow2.sh`: Import existing QCOW2 file to VE

### Data transfer
- `send_file.py`: Sending files over SSH using rsync (incremental)
